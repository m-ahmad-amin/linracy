# Linracy

A full-stack MERN social media platform | Connect, share, and engage with posts, profiles, and interact.
**[▶ Live Demo](https://linracy.netlify.app)**

![Intro](https://res.cloudinary.com/dzzrxqiho/image/upload/v1782030141/Group_5_2_a4puqn.png)

## Overview

Linracy is a Full Stack MERN Social Media Website where users can sign up, create and manage their profiles, and explore other users, along with uploading and browsing photos, all from a responsive and sleek interface.

---

## Features

- **User Authentication**: Secure registration and login system
- **Photo Uploads**: Users can upload and display images
- **Profile Management**: Personalized profile pages with dynamic data
- **Explore Users**: Discover and view other user profiles
- **Responsive UI**: Responsive across mobile, tablet, and desktop
![preview](https://res.cloudinary.com/dzzrxqiho/image/upload/v1782032066/Group_6_4_maoney.png)

## Tech Stack

React.js • Node.js • Express.js • MongoDB • MERN Stack • JavaScript • Cloudinary (Storage) • Deployment on Render and Netlify

## Getting Started

### Prerequisites

- Node.js v18+
- MongoDB (local or Atlas)
- Cloudinary account

### Installation

1. **Clone the repository**

```bash
git clone https://github.com/m-ahmad-amin/linracy.git
cd linracy
```

2. **Install backend dependencies**

```bash
cd Backend
npm install
```

3. **Set up environment variables**


4. **Install frontend dependencies**

```bash
cd ../Frontend
npm install
```

5. **Set up frontend environment variables**


### Running the App

**Start the backend:**

```bash
cd Backend
npm run dev
```

**Start the frontend:**

```bash
cd Frontend
npm run dev
```

The app will be available at `http://localhost:5173`.
